NatCam.RequestBarcode

public static void RequestBarcode (BarcodeRequest[] requests);

Description

Request to be notified when NatCam detects a barcode. Note that when NatCam.Initialize is called, BarcodeDetection must be on. It is off by default.

This is not supported on the Fallback Interface. 

void Start () {
    //Initialize NatCam with barcode detection
    NatCam.Initialize(BarcodeDetection:Switch.On);
    //[Other initialization code...]
    //Start the camera preview
    NatCam.Play();
    //Create an ISBN request
    NatCam.RequestBarcode(new BarcodeRequest(OnDetectedISBN, BarcodeFormat.EAN_13, false));
}

void OnDetectedISBN (List<MachineReadableCode> codes) {
    //Log what we found
    Debug.Log("Found ISBN codes. The first code is " + codes[0].ToString());
}
