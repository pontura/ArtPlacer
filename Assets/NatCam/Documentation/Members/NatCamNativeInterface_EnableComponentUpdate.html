NatCamNativeInterface.EnableComponentUpdate

public static void EnableComponentUpdate (bool state);

Description

Enable or disable raw camera buffer native updates. This is disabled by default. Use this to enable luma (Y channel) and chroma (UV channel) buffer updates in the OnNativePreviewUpdate event. These buffers conform to the YUV 4:2:0 spec. When enabled, these will be propagated before the RGBA buffer handles are propagated. This is because they are gotten directly from the camera. You can send these buffers to your native layer for further processing, just remember to perform a memcpy first as these buffers are intended to be readonly.

If you only need luminance (or a black-and-white preview), you can disable the rendering pipeline and only use the luma channel.

To access this method, uncomment '#define NATCAM_DEVELOPER_MODE' at the top of NatCamNativeInterface.cs.

See Also: NativePreviewCallback , ComponentBuffer .
