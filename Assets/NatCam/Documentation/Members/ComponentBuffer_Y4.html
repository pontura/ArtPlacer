ComponentBuffer.Y4



Description

The luma buffer (Y channel) of the preview. This is a single-channel pixel buffer which corresponds to a MTLR8Unorm texture on Metal or GL_LUMINANCE texture on OpenGLES.

This buffer contains pixel luminance data, and as a result, it is a greyscale version of the camera preview. If this is all you need for your preview processing, you can disable NatCam's rendering pipeline.
