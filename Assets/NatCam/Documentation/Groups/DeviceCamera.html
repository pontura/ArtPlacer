DeviceCamera



Description

This class provides an abstraction for device cameras.



Public Functions

SetFocus Focus the camera to a viewport point.
SetFramerate Set the framerate of the camera to one that satisfies the supplied framerate preset.
SetFramerate Set the framerate of the camera to a supported framerate closest to (framerate).
SetResolution Set the resolution of the camera to one that satisfies the supplied resolution preset.
SetResolution Set the resolution of the camera to a supported resolution closest to (width, height).


Public Variables

ActiveResolution Get the active resolution of the camera.
Facing The facing of the camera.
FlashMode Get or set the camera's flash mode.
FocusMode Get or set the camera's focus mode.
HorizontalFOV The horizontal view angle of the camera in degrees.
IsFlashSupported Whether the camera supports flash.
IsTorchSupported Whether the camera supports torch.
IsZoomSupported Whether the camera supports zooming (ramped or smooth).
TorchMode Get or set the camera's torch mode.
VerticalFOV The vertical view angle of the camera in degrees.
ZoomRatio Get or set the camera's zoom ratio.


Static Variables

Cameras A readonly list of all cameras on the device.
FrontCamera The default front camera of this device.
RearCamera The default rear camera of this device.
